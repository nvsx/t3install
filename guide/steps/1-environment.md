# First Step: Setting up environment

## What we need on our local machine:
- Shell
- Git
- PHP
- Composer
- Docker
- Ddev

## Mac
- Homebrew https://brew.sh/
	- Install Homebrew with Bash command

## Terminal
- Mac recomended: Iterm2
	- $ brew install --cask iterm2
	- or see https://iterm2.com/
- Also recommended: zsh 
	- echo $SHELL
- And also: oh-my-zsh 
	- https://ohmyz.sh/

## Git
- $ which git
- $ brew install git

## PHP
- $ which php
- $ php --version
- $ brew install php

## Composer
- https://getcomposer.org/
- https://getcomposer.org/doc/00-intro.md
- https://tecadmin.net/install-composer-on-macos/
- $ which composer
- $ composer --version

## Docker
- https://docs.docker.com/install/
- $ docker --version

## Ddev
- https://ddev.readthedocs.io/en/stable/
- $ brew install ddev
- $ ddev version
